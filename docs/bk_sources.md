## Текущая боевая конфигурация
- **Кэфы + betable-матчи** → 1xBet (✅ внедрён, бесплатно, headless).
- **Результаты** → HLTV scraper (✅ внедрён, эталон, lock против bo3).
- **Расписание/история** → bo3.gg (бэкбон; bo3-результаты теперь перекрываются HLTV).
